How to Run Locally

To run the FarmIntel application on your own machine:

  1. Clone the repository: git clone https://github.com/sai5056499/FarmIntel.git
  2. Navigate to the project directory: cd FarmIntel
  3. (Optional but Recommended) Create a virtual environment: python -m venv venv
  4. Activate the virtual environment:
    • Windows: .\venv\Scripts\activate
    • macOS/Linux: source venv/bin/activate
  5. Install dependencies: pip install -r requirements.txt (Make sure you have a requirements.txt!)
  6. Navigate to the app folder: cd app
  7. Run the Flask application: python app.py
  8. Open your web browser and go to the address shown in the terminal (usually http://127.0.0.1:5000/).
